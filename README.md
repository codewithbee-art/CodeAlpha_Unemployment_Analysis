# Unemployment Analysis in India using Python

The objective of this project is to analyze unemployment trends in India using Python. The project focuses on understanding unemployment patterns across different states, comparing urban and rural unemployment, analyzing monthly trends, and studying the impact of Covid-19 on unemployment rates.
-----------

## Dataset
The dataset contains unemployment statistics across different regions in India.
Key columns used in this project include:
- State
- Date
- Frequency
- Unemployment Rate
- Employed
- Labour Participation Rate
- Area
-----------

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
----------

## Workflow
The project follows a structured data analysis workflow:

1. Data loading
2. Initial data exploration
3. Data cleaning
4. Date formatting
5. Missing value handling
6. Covid-19 period segmentation
7. State-wise unemployment analysis
8. Urban vs rural unemployment comparison
9. Monthly unemployment trend analysis
10. Seasonal pattern analysis
11. Covid-19 impact analysis
12. Policy insights and conclusion
-----------

## Data Cleaning
The dataset contained extra spaces in column names and some missing values. Column names were cleaned and renamed for better readability. Missing rows were removed to ensure consistency during analysis.
-----------

## Covid-19 Impact Analysis
The dataset was divided into two periods:

- Pre-Covid: before 2020
- During-Covid: 2020 and after

This helped compare unemployment levels before and during the Covid-19 period.
----------

## Visualizations
The project includes visual analysis of:

- Average unemployment rate by state
- Urban vs rural unemployment rate
- Monthly unemployment trend over time
- Seasonal unemployment pattern by month
- Covid-19 impact on unemployment
- Top 5 states with highest unemployment
-----------

## Key Insights
- Unemployment rates vary significantly across states.
- The unemployment rate increased sharply during the Covid-19 period.
- Urban and rural areas show different unemployment patterns.
- Monthly analysis shows fluctuations, but no strong repeating seasonal pattern.
- External shocks such as Covid-19 had a stronger effect on unemployment than normal seasonal changes.
-----------

## Policy Insights
- States with high unemployment may require targeted employment programs.
- Emergency economic support is important during crises such as Covid-19.
- Labour market resilience should be strengthened to reduce the impact of future shocks.
- Data-driven policy decisions can help improve employment planning.
-----------

## Conclusion
This project shows how Python can be used to analyze unemployment trends and generate meaningful economic insights. The analysis highlights regional differences, urban-rural variation, and the significant impact of Covid-19 on unemployment in India.
